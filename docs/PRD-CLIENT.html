1. Visi dan Tujuan Produk

1.1 Pernyataan Visi

CAKRA Intelligence Platform adalah platform operasi terpadu berbasis web yang menyatukan pelacakan geolokasi real-time, komunikasi tim, analisis kecerdasan buatan, dan integrasi video surveillance dalam satu antarmuka — memberikan Common Operating Picture (COP) kepada seluruh pemangku kepentingan operasi.

1.2 Masalah yang Diselesaikan

  • Koordinasi tim lapangan masih menggunakan alat terpisah (WhatsApp, peta kertas, radio) tanpa integrasi data terpusat.
  • Tidak ada visualisasi posisi tim dan aset secara real-time pada satu dashboard.
  • Analisis data operasi (cuaca, medan, riwayat) dilakukan secara manual dan lambat.
  • Aliran video CCTV dan drone tidak terintegrasi dengan peta operasi.

1.3 Tujuan Bisnis

  • Mempercepat waktu respons operasi lapangan hingga 40%.
  • Menyediakan satu sumber kebenaran (single source of truth) untuk seluruh data operasi.
  • Mengurangi ketergantungan pada komunikasi verbal dengan dokumentasi digital otomatis.

4. Alur Pengguna (User Flow)

4.1 Alur Komandan — Memantau Operasi

  1. Login → masuk ke Peta Utama (dashboard)
  2. Melihat posisi seluruh tim lapangan di peta real-time
  3. Aktifkan layer cuaca/topografi untuk analisis medan
  4. Gambar zona pencarian menggunakan drawing tools
  5. Buka tab Percakapan → kirim arahan ke Tim Alpha
  6. Buka tab Asisten AI → tanyakan rekomendasi pola pencarian
  7. Buka tab Prediksi → lihat heat map probabilitas lokasi target
  8. Buka tab Kamera Live → pantau feed CCTV dan drone
  9. Terima notifikasi geofencing → anggota keluar zona

4.2 Alur Operator Lapangan — Mengirim Laporan

  1. Buka aplikasi di browser mobile → login
  2. Posisi GPS otomatis dikirim ke pusat kendali setiap 5–30 detik
  3. Temukan bukti → buka tab Laporan
  4. Pilih kategori (Jejak/Barang/Tanda/Titik Bahaya/Titik Istirahat)
  5. Ambil foto, rekam video atau suara → lampirkan ke laporan
  6. Klik Kirim Laporan → data otomatis muncul di peta pusat kendali
  7. Jika offline → data masuk antrean, terkirim otomatis saat koneksi pulih
  8. Buka tab Percakapan → komunikasi dengan pusat kendali
  9. Tekan tombol Darurat SOS jika diperlukan

4.3 Alur Admin — Mengelola Sistem

  1. Login → masuk ke tab Admin
  2. Lihat daftar pengguna → tambah pengguna baru dengan peran tertentu
  3. Edit atau nonaktifkan pengguna yang tidak aktif
  4. Kelola sumber video (tambah/edit URL stream CCTV/drone)
  5. Unggah dokumen ke basis pengetahuan AI

4.4 Alur Data — Integrasi CCTV dan Drone

  • Menerima aliran video CCTV dari lokasi operasi
  • Menampilkan video streaming CCTV pada titik peta yang relevan
  • Menerima aliran video drone dari lapangan
  • Menampilkan video drone secara real-time pada dashboard
  • Operator dapat memantau dan mengontrol kamera jika didukung
